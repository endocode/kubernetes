## kubectl diff

Show differences between the current configuration of a resource and a configuration file.

### Synopsis


Display any differences between the current configuration of a
resource and a configuration file.

```
kubectl diff (FILENAME)
```

### Examples

```
$ kubectl diff -f my-pod.json
```
